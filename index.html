博客建立的心得（服务器搭建篇）

发表于 2021-09-09 分类于 hexo

博客的建立教程

1、下载Sublime Text

打开博客和GitHub记得开加速器，其他情况建议关闭加速器

1）网址：http://www.sublimetext.com/

已在所给的文件夹压缩包里面下载好了

2）安装过程

除了路径自己可以选择外，其他都是默认，这里就不加以赘述了。

3）博客工程导入

找到sublime_text软件

2、下载next主题

默认主题太难看了，换一个好看的主题

1）网址：https://github.com/theme-next/hexo-theme-next

也帮你下好了，放在压缩包软件文件夹里面

2）next压缩包的使用

3、GitHub当中建立网页文件

互联网在向你招手！！！

1）登录GitHub进入仓库

参考第环境搭建篇“更改默认仓库名字”这一节最后两张图片就可知道进入方法

2）建立HTML文件

只需要知道这是一个给浏览器使用的语言文件

将“用户名.github.io”就会让你仓库里编好的html文件在浏览器运行（相当于跟你博客能放在仓库里让浏览器访问的一个预热）

3）更改博客配置

需要更改相关配置才能准确上传到你的GitHub上（记住betterluo是我的用户名，实际过程中你要用自己的用户名）

4、将你的本地博客文件上传GitHub服务器

1）桌面空白处右击打开Git Bash Here

图中第三个红框命令应该将.email改成.name

2）生成Personal access tokens（私钥）

进入GitHub后进入选项

3）复制过来的Personal access tokens放入配置文件中并保存

4）打开Git Bash Here

5）打开https://betterluo.github.io/

至此你的博客框架已经上传到GitHub当中，并且开加速器能够在客户端互联网上访问，开流量能够在手机浏览器访问，地址都一样。

博客的建立教程

1、搭建框架（记得开加速器）

· node.js安装–搭建JavaScript的运行环境

1）网址链接：https://nodejs.org/zh-cn/

2）等待浏览器下载器下载完成（已下好的软件在本教程的软件文件夹的node-v14.17.6-x64.msi）

3）按照下图操作知道安装完成

4）在cmd窗口当中检查是否安装完毕

打开CMD命令窗口

· github–一个面向开源及私有软件项目的托管平台

网站：https://github.com/

1）按照下图所示，进行注册（相当于购买域名背后的服务器）

2）更改默认仓库名字（也就是生成个人的域名）

· github for windows–通过window系统来管理GitHub仓库

1）下载：https://gitforwindows.org/（已在相应的软件文件夹下载好了）

2）安装过程当中除了更改自己喜欢的路径和下面图片所示选项要求之外，其余都默认就好（一路next直到安装完成）

3）检查是否完成

使用上述方法打开cmd窗口

4）SSH配置

回到桌面，然后鼠标右击空白处，点击Git Bash Here
输入：ssh-keygen -t rsa -C “你的邮箱地址”

成功标志

在保存路径下用记事本打开

在GitHub网页打开复制界面

成功标志

· hexo–静态博客生成器

本地（这个经常用作发送云端之前的预览功能）

1）安装hexo的开源博客软件

输入命令：npm install hexo-cli -g

输入命令：hexo init 博客名

输入命令：npm install

2）打开本地端口命令：hexo s或者hexo s -p 5555(自定义端口)

3）用浏览器打开对应的本地网址

至此，你的博客的框架已经搭建好了
